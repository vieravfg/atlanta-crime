What is our Project about?

Our project focuses on the analysis of Atlanta's crime data from 2009 to 2019 and 2020 to 2021. We will identify crime patterns or trends based on different locations in Atlanta.

  • General overview of crime in Atlanta
  • Correlations between demographics and crime
  • Forecasting of crime counts in Atlanta

Machine Learning

We used Atlanta crime data from 2009 to 2019 to create machine learning models and predict crime count by day. We tested various models, such as AR, MA, ARIMA, ARIMAX, and Prophet. After comparing these different models, we decided to present our analysis with ARIMA and Prophet.

process

  • How did we clean the data?

    Used Jupyter notebook and pandas library to clean and transform the raw data from Atlanta Police website. There were two datasets that had over 300,000 plus rows combined. As part of the cleaning process a few columns were deleted and transformed to have the right data type. Analysis and filters were applied to the dataset to output a final csv file.

  • How did we create our visualizations?

    Used Tableau Public to perform analysis and create visualizations of our dataset.

  • How did we use Machine Learning?

    Used Atlanta's crime data from 2009 to 2019 to create ARIMA and Prophet machine learning models to predict crime counts by day for 2020-01-01 to 2021-12-31. The predicted value saved in MongoDB facilitated a user interface to select a date range in order to see predictions.

  • How did we get our database working?

    We deployed our database by creating a free cluster through a MongoDB cloud platform then created collections using predicted values from machine learning.

  • How did we deploy our website?

    We deployed our website by using Github and Heroku as our hosting platforms. Used HTML, CSS, and JavaScript to create the content and structure of our web pages. Within Heroku, we generated a connection to our MongoDB database for users to interact with predicted values from Machine Learning. The Flask App served to connect our website with the database and share data.

  • How did we interact with the users?

    Users can submit feedback using the message box in our website. This data is collected and transferred to a MongoDB cloud database through the Flask app.
